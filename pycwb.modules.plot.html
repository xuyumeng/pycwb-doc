pycwb.modules.plot.spectrogram.plot_spectrogram(wavearray, xmin=None, xmax=None, figsize=(24, 6), gwpy_plot=True)[source]¶

Plots a spectrogram of the given waveform array

Parameters:

  • wavearray (ROOT.WSeries or gwpy.timeseries.TimeSeries) – given waveform array

  • xmin (float, optional) – x-axis minimum for matplotlib plot

  • xmax (float, optional) – x-axis maximum for matplotlib plot

  • figsize (tuple, optional) – figure size for matplotlib plot, default (24, 6)

  • gwpy_plot (bool, optional) – whether to use gwpy or ROOT to transform the waveform array to a spectrogram, default True

Returns:

the matplotlib plot object for user to further modify or save

Return type:

matplotlib.pyplot
