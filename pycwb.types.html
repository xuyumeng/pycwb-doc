class pycwb.types.wdm.WDM(m=None, k=None, beta_order=None, precision=None, wavelet=None)[source]¶

Bases: object

Wrapper for ROOT.WDM object

after forward transformation the structure of the WDM sliced array is the following:

f phase 0 phase 90

M * * … * * * … *

… … … … … … …

1 * * …. * * * … *

0 * * … * * * … *

t 1 2 … n n+1 n+2… 2n

where t/f is the time/frequency index, the global TF index in the linear array is i = t*(M+1)+f

Parameters:

  • m (int) – number of bands

  • k (int) – k=n*m, where n is integer. K defines the width of the ‘edge’ of the basis function in Fourier domain (see the paper). larger n, longer the WDM filter, lower the spectral leakage between the bands.

  • beta_order (int) – defines the sharpness of the ‘edge’ of the basis function in Fourier domain (see paper)

  • precision (int) – defines filter length by truncation error quantified by P = -log10(1 - norm_of_filter) (see the paper)

  • wavelet (ROOT.WDM, optional) – ROOT.WDM object, if provided, other parameters are ignored and extracted from the object
